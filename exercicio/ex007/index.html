Texto marcado

Podemos criar um texto marcado com a tag MARK

Texto deletado

Podemos marcar um texto deletado com a tag DEL , para indicar que deve ser lido mas não considerado

Texto inserido

Podemos marcar um texto inserido, para indicar que o texto foi adicionado depois com a tag

Texto sobrescrito

Para inserir coisas acima de outras por exemplo: x20+3

Texto subscrito

Para inserir coisas abaixo de outras por exemplo: H2O
